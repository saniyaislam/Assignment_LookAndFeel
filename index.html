About Assignment

  • Responsiveness

    The layout of page has been made flexible to fit the visitor's screen size and change accordingly. We have used media queries to implement responsiveness. The higher range of the width of screen is set as 768px as it is the usual size of tablet displays. For mobile and other small devices, we have adjusted the float pixels. Hence, website is made compatible with small (mobile), medium (tablet) and large (desktop) screen sizes.

  • Usability

    Our website has a simple layout with familiar tabs. Consistency is maintained across all the 4 navigation pages which are present at the top right corner. The page displays its purpose very clearly and all pages are accessible with no broken links. Home page displays the description of website. Menu page displays the menu of restaurant. Reservation page take reservations from visitors. About project page explains the design and other parameters of the website.

  • Content

    Our website contains relevant content. As it is a restaurant website, so text is written in a way it catches the search for restaurants. Also, the visual material like Images and graphics supports the text in website and therefore increases the relevance with greater impact.

  • Design

    The website is designed using HTML5 CSS3 tags. To group different sections of the screen, various div classes and IDs are used in an assignment. This helps in styling each page effectively. Float in CSS is used to style the text content align an spread the content in website. This has made our website more presentable. Media queries are used to create responsiveness in website with flexible layout
